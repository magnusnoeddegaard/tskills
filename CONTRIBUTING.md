# Contributing to skillsync

Thanks for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/skillsync.git`
3. Install dependencies: `npm install`
4. Build: `npm run build`
5. Link for local testing: `npm link`

## Development

```bash
npm run build    # Compile TypeScript
npm run dev      # Watch mode
```

## Making Changes

1. Create a branch: `git checkout -b feature/my-feature`
2. Make your changes
3. Test locally with `ss` commands
4. Commit with a clear message
5. Push and open a Pull Request

## Pull Request Guidelines

- Keep PRs focused on a single feature or fix
- Update README.md if adding new commands or changing behavior
- Add yourself to contributors if this is your first PR

## Code Style

- TypeScript with strict mode
- Use async/await over callbacks
- Keep functions small and focused

## Reporting Issues

- Check existing issues first
- Include your OS, Node version, and steps to reproduce
- For bugs, include error messages and expected vs actual behavior

## Questions?

Open an issue with the "question" label.
